# Nunik Co. | High-End Digital Collective
> **Precision-Engineered Brand Identities & Intelligent Systems.**

Nunik Co. is a premium agency at the intersection of technical mastery and minimalist aesthetics. We specialize in **"Yukti" (Strategic) Automation**, high-performance web architecture, and elite brand identity.

This repository contains the source code for the Nunik Co. flagship experience—a WebGL-powered, glassmorphic journey designed for the forward-thinking elite.

## 🛠 High-Performance Stack
- **Framework:** Pure Vanilla HTML5 / CSS3 / ES6 JavaScript
- **Animation Engine:** GSAP + ScrollTrigger
- **Smooth Scroll:** Lenis by Studio Freight
- **Visuals:** Pure Three.js WebGL (Custom Shaders)
- **Styling:** Tailwind CSS 4 (via CDN) + Pure CSS Variables

## 💎 Design Philosophy & UI Tokens
- **Absolute Minimalism:** Generous whitespace (60% of viewport empty at all times).
- **Translucent Hierarchy:** All UI elements are frosted glass (`backdrop-filter: blur(24px)`), allowing the 'Soul' (WebGL Gradients) to bleed through.
- **Tactile UI:** Strict adherence to 8px border radiuses and soft, diffused drop shadows (0 8px 32px rgba(0,0,0,0.2)) to elevate glass layers.
- **Kinetic Interaction:** Every letter and scroll move reacts to the user. Hover states trigger subtle vertical lifts and increased translucency.
